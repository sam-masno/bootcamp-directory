# Coding Bootcamp Directory API

Backend API for DevCamper application, which is a bootcamp directory website

Usage
Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies
```npm install```

## Run App
rename config/config.env.env to config/config.env

Inside the new config/config.env replace environment variables to your own.

## Run in dev mode
```npm run dev```

## Run in prod mode
```npm start```

To seed the database with users, bootcamps, courses and reviews with data from the "_data" folder, run the following in root folder

### Destroy all data
```node seeder -d```

### Import all data
```node seeder -i```
